# Project-Afternight-Roblox-Autoplay-WIP-

Used for the game Project: Afternight to autoplay. Im also sure it would work on others with a bit of configuration.
This code works for middlescroll + downscroll with basic arrow skins and a 1920 x 1080 resolution, however, feel free to make changes to the resolution or whatever so that it fits your liking.
I assume it also doesnt work on songs with excessive camera shaking.

--- Note keys ---
Left = D
Down = F
Up = J
Right = K
